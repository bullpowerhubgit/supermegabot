# My-Shop Frontend

React + TypeScript + Vite Dashboard fuer SuperMegaBot E-Commerce.

## Schnellstart

```bash
cd my-shop/frontend
npm install
npm run dev
```

## Features

- **Dashboard** - Umsatz, Besucher, Konversionen
- **Produkte** - Inventarverwaltung mit Filtern
- **Bestellungen** - Auftragsstatus-Tracking
- **Marketing** - Kampagnen & Performance
- **SEO** - Rankings & technische Analyse

## Integration

Verbindet sich automatisch mit `my-shop/backend` (Port 4000).
