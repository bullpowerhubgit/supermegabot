## Quick Deployment Guide

### 1. Vercel Deployment

```bash
# Install Vercel CLI
npm i -g vercel

# Login to Vercel (Team Account)
vercel login

# Deploy to Team
vercel --team team_xulvdt7sib2RSt4BNoqVWeSy
```

### 2. Environment Variables

Setze im Vercel Dashboard:

- `ANTHROPIC_API_KEY` - Claude API Key
- `NODE_ENV=production`

### 3. Local Development

```bash
npm install
npm run dev
# Visit http://localhost:3000
```
