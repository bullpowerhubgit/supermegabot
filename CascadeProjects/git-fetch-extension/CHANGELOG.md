# Change Log

## [1.0.0] - 2024-01-01

### Added
- Initial release of Git Fetch Extension
- Smart Git Detection for workspace folders
- Automatic periodic git fetch with configurable interval
- Status bar indicator showing repository count and last fetch time
- Manual control commands (Start, Stop, Manual Fetch)
- Multi-workspace support
- Optional completion notifications
- Error handling for non-git folders and fetch errors
- Dynamic workspace adaptation
