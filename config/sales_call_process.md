## Sales-Call Prozess (Standard — 20–30 Min)

### 0. Qualifizierung (vor dem Call)
- Shop / Business-Typ (Shopify, Digistore, Agency, Solo)
- Monatlicher Umsatz-Band (0–5k / 5–30k / 30k+)
- Haupt-Schmerz: Support · Traffic · Conversion · Zeit
- Budget-Fit: Starter €49 / Pro €99 / Scale €299

### 1. Opening (2 Min)
- Agenda: Ist-Zustand → Demo/Case → Fit → Next Step
- Erlaubnis: „Darf ich 3 Fragen stellen?“

### 2. Discovery (8 Min)
1. Was läuft manuell und kostet die meiste Zeit?
2. Wo liegt der größte Umsatz-Leak (Checkout, Ads, Support)?
3. Was wäre in 30 Tagen ein Erfolg?
4. Wer entscheidet / wann startbereit?

### 3. Case Study (5 Min)
- 1–2 passende Cases (siehe unten) mit **Messzahl + Zeitraum**
- „Ähnlich wie bei [Case]: Ergebnis X in Y Wochen“

### 4. Demo / Solution Map (7 Min)
- Zeige Dashboard / Bot / Automation die den Schmerz löst
- Max 3 Features — Rest Follow-up

### 5. Close (5 Min)
- **Option A (Sofort):** Starter €49 / 7-Tage Trial → Stripe-Link im Chat
- **Option B (High-Touch):** Pro/Scale + Setup → Stripe + Onboarding-Call
- **Option C (Nein/Später):** Case-PDF + Follow-up in 48h

### 6. Nach dem Call (automatisch)
- T+0: Danke + Link + Case Summary
- T+24h: Follow-up 1 (Value)
- T+72h: Follow-up 2 (Close / Last call)

## Case Studies (Kern — übergreifend)

### CS1 — Shopify Support + Recovery
- **Persona:** Shopify Solo / DTC
- **Ausgang:** Manuelle FAQs, Abandoned Checkouts ungenutzt
- **Ergebnis:** −40% Support-Zeit, +€2.1k Recovery / 45 Tage (Beispiel-Setup)
- **Pitch-Zeile:** „Weniger Tickets, mehr Checkout-Nachfassen — ohne neues Team.“

### CS2 — Multi-Tool Hub (Agency)
- **Persona:** Agency / Multi-Shop
- **Ausgang:** 9 Shops, fragmentierte Tools
- **Ergebnis:** +Ops-Effizienz, 1 Command Center, Fokus Closing
- **Pitch-Zeile:** „Ein Hub statt 12 Logins — Team verkauft statt firefighting.“

### CS3 — Telegram + Stripe Subscription
- **Persona:** Creator / Community
- **Ausgang:** Kostenlose Gruppe, kein recurring
- **Ergebnis:** Paid Channel + Auto-Zugang nach Zahlung
- **Pitch-Zeile:** „Follower → Abo in einem Bot-Flow.“

### CS4 — Steuern / Compliance (DACH)
- **Persona:** Online-Händler AT/DE
- **Ausgang:** USt/OSS Chaos
- **Ergebnis:** Stunden → Minuten, weniger Nachzahlungsrisiko
- **Pitch-Zeile:** „Compliance im Autopilot — du prüfst nur noch.“
