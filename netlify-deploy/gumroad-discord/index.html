SETUP GUIDE

In 4 Schritten live gehen

Keine Programmierkenntnisse nötig. Von Einrichtung bis live in unter 10 Minuten.

1
Bot zu deinem Server hinzufügen
Klicke auf "Bot hinzufügen", wähle deinen Server und bestätige die Berechtigungen. Der Bot braucht: Rollen verwalten, Nachrichten senden, Kanäle lesen.
🤖 GumroadBot wurde deinem Server hinzugefügt! Tippe /setup um zu beginnen.
2
Gumroad Webhook verbinden
Kopiere deine Webhook-URL aus dem Bot-Dashboard und füge sie in deinen Gumroad-Produkt-Einstellungen unter "Ping-URL" ein.
/setup webhook
📋 Deine Webhook-URL:
https://gumroadbot.app/wh/xyz123
Füge diese URL in Gumroad ein ✓
3
Premium-Rolle konfigurieren
Sag dem Bot welche Discord-Rolle er nach einem Kauf vergeben soll. Du kannst verschiedene Rollen für verschiedene Produkte einrichten.
/role set @Premium KI-Kurs-2026
✅ Konfiguriert: Produkt "KI-Kurs-2026" → Rolle @Premium
Tipp: Setze auch /role set @VIP für andere Produkte
4
Fertig — Test machen
Kaufe dein eigenes Produkt einmal mit einer Test-E-Mail und beobachte wie der Bot in weniger als 3 Sekunden die Rolle vergibt.
/test verify test@email.com
Simuliere Kauf für test@email.com...
🎉 Test erfolgreich! Rolle wurde vergeben in 1.8 Sekunden
