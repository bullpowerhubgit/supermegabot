# RudiBot Browser Extension — Installation

## In Chrome/Brave/Arc installieren:

1. Öffne Chrome → Adressleiste: `chrome://extensions/`
2. Oben rechts: **"Entwicklermodus"** aktivieren
3. Klicke **"Entpackte Erweiterung laden"**
4. Wähle diesen Ordner: `/Users/rudolfsarkany/supermegabot/browser-extension`
5. ✅ RudiBot erscheint in der Chrome-Toolbar!

## Verwendung:
- **Klick auf 🤖** in der Toolbar → Chat-Popup
- **Cmd+Shift+R** → Widget auf jeder Website öffnen
- **Rechtsklick** auf Text → "RudiBot analysieren"

## Schnellbefehle im Popup:
- 📊 Bot Status
- 🛒 Shopify
- 📱 Social Media
- 💰 Einnahmen
- 🔍 Seite analysieren
- 📤 Auto-Post

## Voraussetzung:
Bot muss laufen auf http://localhost:3200
→ Öffne RudiBot.app auf dem Desktop
